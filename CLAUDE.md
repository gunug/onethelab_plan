# Claude.md - 프로젝트 변경 기록

## 프로젝트 규칙

- 사용자 요청이 있을 때마다 프로젝트 내용이 바뀔 때마다 이 파일(claude.md)에 기록할 것
- 변경 사항은 날짜, 변경 내용, 관련 파일을 포함하여 기록
- **plan_obsidian**: 옵시디언 유형의 마크다운 파일을 작성하는 폴더
  - 앞으로 md 문서는 이 폴더 안에 작성
  - 폴더 구조를 claude.md에 기록하고 관리
