# 배너 광고에 적합한 게임 장르 분석

> **핵심 질문**: 배너 광고만으로 수익이 유효한 게임 장르는 무엇인가?

## 결론 요약

| 장르 | 배너 광고 적합도 | 수익 잠재력 | 추천도 |
|------|------------------|-------------|--------|
| **아이들/클리커** | ⭐⭐⭐⭐⭐ | 중 | 🥇 1위 |
| **퍼즐 (스도쿠, 십자말)** | ⭐⭐⭐⭐⭐ | 중 | 🥇 1위 |
| **단어 게임** | ⭐⭐⭐⭐ | 중 | 🥈 2위 |
| **캐주얼/하이퍼캐주얼** | ⭐⭐⭐ | 낮음 | 🥉 3위 |
| **시뮬레이션** | ⭐⭐⭐ | 중 | 4위 |
| **전략/RPG** | ⭐⭐ | IAP 위주 | 5위 |
| **MMORPG** | ⭐ | IAP 위주 | 비추천 |

## 배너 광고의 특성

### 장점
- **비침입적**: 게임플레이 방해 최소화
- **지속적 노출**: 화면에 항상 표시되어 꾸준한 수익
- **구현 간편**: 가장 쉽게 적용 가능한 광고 유형
- **유저 이탈 적음**: 전면 광고보다 거부감이 낮음

### 단점
- **낮은 eCPM**: $0.10 ~ $2.00 수준 (보상형 동영상의 1/10 이하)
- **배너 무시 현상**: 유저들이 배너를 무시하는 경향 증가
- **낮은 CTR**: 클릭률이 다른 광고 유형보다 현저히 낮음

### 광고 유형별 eCPM 비교 (미국 기준)

| 광고 유형 | eCPM 범위 | 비고 |
|-----------|-----------|------|
| 배너 | $0.10 ~ $2.00 | 가장 낮음 |
| 전면 광고 | ~$11 | Android 기준 |
| 보상형 동영상 | ~$13 ~ $20 | 완료율 75~95% |
| 플레이어블 | $10 ~ $30 | 설치 의도 높음 |
| 오퍼월 | ~$400 (최대 $1,500) | 가장 높음 |

## 장르별 상세 분석

### 🥇 1위: 아이들/클리커 게임

**배너 광고 적합도: ⭐⭐⭐⭐⭐**

**특징:**
- 단순한 게임 메카닉으로 화면 공간 확보 용이
- 긴 플레이 세션 (평균 25분/일, 하이퍼캐주얼의 2.5배)
- 높은 세션 빈도 (하루 평균 4회 이상)
- 짧은 개별 세션 (3분 미만)이지만 자주 방문

**수익 구조:**
- 광고 수익 60~70%, IAP 30~40%
- 배너 광고가 "거의 눈에 띄지 않지만 추가 수익 창출"
- 대부분의 아케이드 아이들 게임이 배너 광고 사용

**적합한 이유:**
```
✅ 단일 화면 구성 → 배너 배치 자연스러움
✅ 자주 방문 → 노출 수 극대화
✅ 게임플레이 단순 → 배너가 방해되지 않음
✅ 캐주얼 유저층 → 광고 거부감 낮음
```

**주의사항:**
- 보상형 동영상이 핵심 수익원 (약 80%)
- 배너는 "보조 수익원"으로 활용

---

### 🥇 1위: 퍼즐 게임 (스도쿠, 십자말, 블록 퍼즐)

**배너 광고 적합도: ⭐⭐⭐⭐⭐**

**특징:**
- 긴 플레이 세션 (한 문제에 5~30분)
- 화면 변화가 적음 → 배너 위치 고정 가능
- 2024년 퍼즐 장르 매출 $12.2B (전체 3위)
- 광고 의존도 ~80%로 매우 높음

**수익 구조:**
- 블록 퍼즐 하위 장르 +91% 성장 (2024-2025)
- 전체 퍼즐 시장 +14% 성장
- IAA(인앱광고)가 수익의 대부분 차지

**적합한 이유:**
```
✅ 긴 세션 → 배너 노출 시간 극대화
✅ 집중형 게임 → 화면 하단 배너 방해 적음
✅ 광고 의존도 높은 장르 → 유저 수용도 높음
✅ 심플한 UI → 배너 배치 용이
```

**Google AdMob 권장:**
> "십자말풀이 또는 스도쿠 게임의 경우 배너 광고도 흔히 쓰입니다."

---

### 🥈 2위: 단어 게임

**배너 광고 적합도: ⭐⭐⭐⭐**

**특징:**
- 퍼즐과 유사한 플레이 패턴
- 생각하는 시간 동안 배너 노출
- 상대적으로 단순한 UI

**수익 구조:**
- 전면 삽입 광고 + 배너 광고 조합 일반적
- 퍼즐 장르와 함께 주요 광고 수익 장르로 분류

---

### 🥉 3위: 캐주얼/하이퍼캐주얼

**배너 광고 적합도: ⭐⭐⭐**

**특징:**
- 짧은 세션 (평균 10분/일)
- 매우 높은 세션 빈도
- 낮은 LTV, 높은 볼륨

**수익 구조:**
- 전면 광고 + 보상형 동영상 위주
- 배너는 보조적 역할

**주의사항:**
- 화면이 자주 바뀌는 게임에서는 배너 효과 감소
- 전면 광고가 더 효과적인 장르

---

### 4위: 시뮬레이션

**배너 광고 적합도: ⭐⭐⭐**

**특징:**
- 몰입도가 중요한 장르
- 네이티브 광고가 더 적합

**수익 구조:**
- IAP 62%, 광고 38%로 IAP 비중 높음
- 배너보다는 네이티브/프로그래매틱 광고 권장

---

### 5위: 전략/RPG

**배너 광고 적합도: ⭐⭐**

**특징:**
- IAP 중심 수익 모델 ($17.5B, 1위)
- 복잡한 UI로 배너 배치 어려움
- 코어 유저층은 광고 거부감 높음

**수익 구조:**
- 전체 매출의 21.4%가 전략 장르
- 다운로드 비중은 4%에 불과 → 고과금 유저 타겟

---

### 비추천: MMORPG

**배너 광고 적합도: ⭐**

**이유:**
- 몰입도가 핵심 → 광고가 경험 해침
- IAP 중심 수익 모델
- 복잡한 UI와 긴 세션

> "몰입도가 높은 MMORPG의 특성상 인앱 광고가 다른 장르보다 어렵습니다." - Google AdMob

## 배너 광고 수익 극대화 전략

### 1. 배너 + 보상형 동영상 하이브리드

```
배너: 기본 수익층 (꾸준한 소액)
보상형: 핵심 수익원 (고eCPM)
전면광고: 레벨 전환 시점
```

### 2. 적합한 배너 위치

| 게임 유형 | 권장 위치 |
|-----------|-----------|
| 아이들 게임 | 화면 하단 고정 |
| 퍼즐 게임 | 문제 풀이 영역 외부 (상단/하단) |
| 단어 게임 | 키보드 위 또는 화면 상단 |

### 3. 세션 기반 최적화

- **긴 세션 게임**: 배너 효과적
- **짧은 세션 게임**: 전면 광고가 더 효과적
- **자주 방문하는 게임**: 배너 노출 누적 효과

## 최종 권장사항

### 배너 광고가 "유효한" 장르

1. **아이들/클리커** - 보조 수익원으로 최적
2. **퍼즐 (스도쿠, 십자말풀이, 블록)** - 긴 세션으로 효과적
3. **단어 게임** - 퍼즐과 유사한 패턴

### 배너 광고 "단독"으로 수익화 가능한가?

**결론: 매우 어려움**

- 배너만으로는 CPI 회수 어려움
- 반드시 보상형 동영상/전면 광고와 병행 필요
- 배너는 "기본 수익층"으로 활용

### 실질적인 조합 전략

```
🎯 아이들/퍼즐 게임 추천 광고 믹스:
- 배너: 화면 하단 상시 노출 (20% 수익)
- 보상형: 진행 가속, 힌트 획득 (60% 수익)
- 전면: 레벨/스테이지 전환 시 (20% 수익)
```
