# 배너 광고 CPI와 LTV 분석

## 개요

배너 광고만으로 수익을 낼 수 있는가? 보상형 비디오나 전면광고 대비 낮은 eCPM을 가진 배너 광고의 수익성 분석.

---

## 배너 광고 기본 정보

### 배너 광고란?

- 화면 상단/하단에 **상시 노출**되는 작은 광고
- 유저 행동을 방해하지 않음 (비침습적)
- 클릭 없이도 노출만으로 수익 발생 (CPM)
- Fill Rate 90%+ (거의 항상 광고 노출 가능)

### 광고 유형별 eCPM 비교 (미국, Android, Q4 2024)

| 광고 유형 | eCPM | 배너 대비 |
|-----------|------|----------|
| 오퍼월 (Offerwall) | $400.00 | 588배 |
| 보상형 비디오 | $16.49 | 24배 |
| 전면광고 (Interstitial) | $14.08 | 21배 |
| 네이티브 광고 | $3.00~$3.30 | 4.4~4.9배 |
| **배너 광고** | **$0.68** | **1배 (기준)** |

> 출처: [Tenjin - Ad Monetization Benchmark Report 2025](https://tenjin.com/blog/ad-mon-gaming-2025/)

## 배너 광고 수익 계산 기초

### 핵심 공식

```
배너 수익 = 노출 수 × (eCPM ÷ 1000)
```

| 미국 (iOS) | $0.45 | **$0.00045** |
| 글로벌 평균 | $0.50 | **$0.00050** |

**CPI $1.50 회수 조건:**

```
필요 노출 수 = CPI ÷ 배너 1회 수익
            = $1.50 ÷ $0.00068
            = 2,206회
```

## 배너 광고의 장단점

### 장점

| 장점 | 설명 |
|------|------|
| **높은 Fill Rate** | 90%+ (거의 항상 광고 표시) |
| **비침습적** | 유저 경험 방해 최소화 |
| **상시 수익** | 플레이 중 지속적으로 수익 발생 |
| **구현 간단** | 복잡한 로직 불필요 |
| **이탈 없음** | 광고로 인한 앱 종료 위험 낮음 |

### 단점

| 단점 | 설명 |
|------|------|
| **매우 낮은 eCPM** | 다른 유형 대비 1/20~1/25 수준 |
| **클릭률 저조** | 0.1% 미만 (대부분 무시) |
| **화면 공간 차지** | 게임 UI 제약 |
| **단독 수익화 불가** | CPI 회수가 사실상 불가능 |

## 결론: 배너 광고만으로 수익이 가능한가?

### 수학적 결론

```
[배너 광고만 사용 시]
- CPI $1.50 회수에 필요한 플레이 시간: 18.4시간
- 하이퍼캐주얼 평균 총 플레이 시간: 45분~2시간
- 결과: CPI의 4~10%만 회수 → 90%+ 적자

[현실적 결론]
- 배너 광고 단독 수익화는 구조적으로 불가능
- 배너는 "베이스 수익"으로만 활용
- 전면광고, 보상형 비디오가 메인 수익원이어야 함
```

### 배너 광고의 적절한 역할

| 역할 | 설명 |
|------|------|
| **베이스라인 수익** | 광고 없는 시간에도 최소 수익 확보 |
| **Fill Rate 보완** | 보상형/전면 광고가 없을 때 대체 |
| **비결제 유저 수익화** | IAP 안 하는 유저에게서 최소 수익 |
| **UX 친화적 옵션** | 광고 싫어하는 유저에게 덜 침습적 선택 |

### 권장 광고 믹스

| 광고 유형 | 수익 기여도 | 역할 |
|-----------|-----------|------|
| 보상형 비디오 | 50~60% | 메인 수익 |
| 전면광고 | 30~40% | 보조 수익 |
| 배너 | 5~15% | 베이스 수익 |
